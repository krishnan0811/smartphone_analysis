#  Smartphone Performance Analysis Dashboard

##  Overview
This project presents an interactive dashboard that analyzes smartphone performance data.  
It provides insights into key metrics such as RAM capacity, processor distribution, battery performance, and brand-wise comparisons to understand market trends and device performance.

---

##  Objectives
- Analyze smartphone performance across different brands  
- Compare RAM and price relationships  
- Study processor distribution among devices  
- Evaluate battery capacity by brand  
- Identify top-performing smartphone brands  

---

##  Dataset
The dataset includes detailed smartphone specifications such as:

- Brand Name  
- RAM Capacity  
- Price  
- Processor Type  
- Battery Capacity  
- Camera Resolution  
- Screen Resolution  

---

##  Tools & Technologies
- Power Bi  
- Excel / CSV  

---

##  Dashboard Features
-  RAM vs Price Analysis (Scatter Plot)  
-  Processor Brand Distribution (Donut Chart)  
-  Battery Capacity by Brand (Bar Chart)  
-  Top Brands by Camera Resolution  
-  Screen Resolution Comparison  
-  Interactive Filters for better analysis  

---

##  Key Insights
- Higher RAM devices generally have higher prices  
- Snapdragon processors dominate the market  
- Some brands offer higher battery capacity at lower prices  
- Camera resolution varies significantly across brands  
- Mid-range devices show balanced performance  

---

# Recommendations
- Focus on optimizing performance in mid-range devices  
- Improve battery efficiency in high-performance phones  
- Balance pricing with hardware specifications  
- Enhance camera features for competitive advantage  

---

##  Challenges
- Data consistency and formatting  
- Handling missing values  
- Designing clear and meaningful visualizations  

---

##  Conclusion
The dashboard helps in understanding smartphone performance trends and supports better decision-making for consumers and businesses.

---

##  Future Enhancements

- Integrate advanced analytics and predictions
